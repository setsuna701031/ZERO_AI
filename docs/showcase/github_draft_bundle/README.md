# GitHub Draft Bundle (Showcase)

This folder contains example outputs generated by ZERO.

ZERO does NOT push to GitHub automatically.

Instead, it generates a complete draft bundle:

- commit_message.txt
- pr_description.md
- devlog_entry.md
- publish_plan.md

These files are:

- Fully local
- Human-reviewable
- Safe (no external side effects)

This demonstrates controlled automation, not autonomous execution.
