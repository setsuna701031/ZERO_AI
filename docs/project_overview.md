\# ZERO AI Project Overview



\## 專案名稱

ZERO AI / zero\_ai



\## 專案定位

本專案的目標是建立一套本地執行的 AI 平台雛形，逐步從工具集合發展成可擴充的 AI 工程助手。



\## 核心方向

\- 本地執行為主

\- 先建立可控骨架，再逐步增加智能能力

\- 優先完成 CLI、API、Router、Tools、Memory 等基礎結構

\- 後續再接入本地模型、Planner、Executor、GUI、語音、視覺等模組



\## 當前階段

目前屬於 AI 平台骨架早期階段，重點在：

\- 建立 CLI 控制能力

\- 建立 Flask API 對外接口

\- 打通最小可用請求流程

\- 將對話中的架構知識逐步搬回本地文件



\## 設計原則

\- 優先穩定與可理解

\- 先小後大，避免一開始堆太多功能

\- 模組邏輯盡量拆分，不把所有東西塞進單一檔案

\- 所有重要進度與架構決策，應逐步沉澱到本地 docs
